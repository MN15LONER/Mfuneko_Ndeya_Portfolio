Mzansi-React-App

Mzansi React is a smart shopping platform built with React that aggregates prices from multiple South African retailers across food, electronics, and clothing. Users can browse deals, place orders, and communicate directly with delivery drivers — all within the app. The admin dashboard allows store managers to post leaflets and promotions in real time. By centralizing store data from brands like Woolworths, Shoprite, Mr Price, and more, the app solves the hassle of comparing prices across malls and websites, offering users the best deals from one convenient interface.
